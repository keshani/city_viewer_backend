TO RUN PROJECT
1. DOWNLOAD THE PROJECT FROM GITHUB
    URL: https://github.com/keshani/city_viewer_backend
2. GO TO THE ROOT FOLDER OF THE PROJECT THROUGH COMMANLINE
      EX:../Documents/gitHub/city_viewer_backend
3. RUN BELOW COMMAND TO RUN SPRING BOOT APPLICATION IN PORT 9092
     ./mvnw spring-boot:run
